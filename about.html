About This Project

This is a Cloud Computing practical lab assignment. The goal is to build and deploy a multi-page website using GitHub Pages as the cloud hosting platform.

What I Learned

  • Creating multi-page HTML websites
  • Linking pages with navigation
  • Styling with external CSS files
  • Hosting on GitHub Pages
  • Git commit and push workflow

Tech Stack

  • HTML5
  • CSS3
  • GitHub Pages (Cloud Hosting)
  • Git Version Control

About the Lab

Lab Task 2 focuses on enhancing a cloud-hosted website with CSS styling and multi-page navigation. It builds on Task 1 by adding design and structure to a live deployment.

Deployment

The website is live via GitHub Pages at username.github.io/mywebsite. All changes pushed to the main branch are automatically deployed.
